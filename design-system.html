03 — Application

Uygulama, dört durumda

macOS pencere chrome'unun içinde bütün sistem bir araya geliyor. Idle → Running → Done; ayrıca Error.

Running — ana durum Tek pencere, sol kontrol rayı, ana akış, sağ aşama/log paneli.
Done — transkript hazır
Error — kullanıcı dostu hata
Idle — başlamadan önce
